Вопросы

  1. Какие есть способы объявления функций?
    ОТВЕТ:1. Декларация, 2. Анонимное Выражение, 3. Стрелочные (Arrow), 4. IIFE
  2. Приведите примеры вызова одной и той же функции всеми известными вам способами?

  3. В чём разница между тестированием и отладкой (дебаггингом)? А что такое логирование?
    ОТВЕТ: Тестирование - проверка приложения на корректность работы в различных условиях. Отладка - устранение ошибок работы приложения. Логирование - ведение журнала состояния ключевых параметров в процессе работы приложения
  4. В чём разница между Function Expression и Function Declaration?
    ОТВЕТ: Декларация начинается со слова Function, за которым идёт имя функции, список аргументов в скобках и реализация в фигурных скобках.

    Function Expression - объявление функции в выражении присваивания. Функция может быть вызвана через имя переменной, которой она присвоена. Чаще всего Function Expression анонимные - между function и скобками отсутствует имя. При объявлении таких функций имя можно прописать, но оно будет доступно только внутри функции

    Функции, объявленные через декларацию доступны в любой части кода, через выражение - только в потоке кода после выполнения этого выражения

    See the Pen Декларация функций by Анастасия (@sowlushka) on CodePen.

  5. Что делает функция console.log()?

    ОТВЕТ: Выводит строку в консоль, возвращает undefined
  6. По какой причине, мы можем использовать до объявления функцию, объявленную через Function Declaration, но не можем ту, которая объявлена через Function Expression ?
    ОТВЕТ: Декларации в памяти движка появляются при компилировании кода. Через выражения присваивания ссылки на функцию появляются в момент присваивания.
  7. Что такое глобальная зона видимости функций?
    ОТВЕТ: Зона видимости на уровне модуля, доступная внутри функций и блоков
  8. Что вернёт код?
    function showX (x)
    {
      return x;
    }
    console.log(showX(28));
