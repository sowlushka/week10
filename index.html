  3. В чём разница между тестированием и отладкой (дебаггингом)? А что такое логирование?
    ОТВЕТ:Тестирование - проверка приложения на корректность работы в различных условиях. Отладка - устранение ошибок работы приложения. Логирование - ведение журнала состояния ключевых параметров в процессе работы приложения
  4. В чём разница между Function Expression и Function Declaration?

    ОТВЕТ: Декларация начинается со слова Function, за которым идёт имя функции, список аргументов в скобках и реализация в фигурных скобках.

    Function Expression - объявление функции в выражении присваивания. Функция может быть вызвана через имя переменной, которой она присвоена. Чаще всего Function Expression анонимные - между function и скобками отсутствует имя. При объявлении таких функций имя можно прописать, но оно будет доступно только внутри функции

    See the Pen Декларация функций by Анастасия (@sowlushka) on CodePen.
